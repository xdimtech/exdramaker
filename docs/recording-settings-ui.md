# 录制设置页面 UI 设计文档

## 整体布局

模态对话框，白色背景，大圆角（约 16px），外层有深色半透明遮罩。整体为**左右双栏布局**：

- **左栏**：固定的预览区域（约占 35% 宽度）
- **右栏**：可滚动的设置面板（约占 65% 宽度）

```
┌──────────────────────────────────────────────────────────┐
│  录制设置                                          [×]   │
├──────────────────────────────────────────────────────────┤
│              │  画面比例                                  │
│   预览       │  背景                                      │
│  ┌────────┐  │  圆角半径                                  │
│  │        │  │  摄像头                                    │
│  └────────┘  │  麦克风                                    │
│              │  画布边距                                  │
│              │  鼠标光标效果                              │
│              │  账户                                      │
├──────────────────────────────────────────────────────────┤
│  [☆ 去除水印 — $20 一次性付款]          [    完成    ]   │
└──────────────────────────────────────────────────────────┘
```

---

## 各区域详细说明

### 头部

| 元素     | 描述                                |
| -------- | ----------------------------------- |
| 标题     | "录制设置"，左对齐，约 20px 加粗    |
| 分隔线   | 标题下方有一条细灰线                |
| 关闭按钮 | 右上角 "×"，圆形浅灰色背景，约 24px |

---

### 左栏：预览区域

- 标签 "预览" 置于预览框正上方，小号灰色文字
- 预览框为白色矩形，带有**渐变彩色边框**（顶部/右侧为粉紫色渐变，底部为绿色渐变）
- 预览内容：
  - 三条水平占位线（模拟文字内容）
  - 一个粉红色实心圆（模拟摄像头画面位置）
  - 右下角一个灰色圆（模拟光标高亮效果）
- 预览框比例随"画面比例"设置实时联动

---

### 右栏：设置面板

#### 1. 画面比例

6 个选项，排列为 **3 列 × 2 行**的按钮组：

| 行  | 列 1                       | 列 2       | 列 3          |
| --- | -------------------------- | ---------- | ------------- |
| 1   | **16:9 YouTube**（已选中） | 4:3 经典   | 3:4 小红书    |
| 2   | 9:16 抖音                  | 1:1 正方形 | Custom 自定义 |

**样式规范：**

- 每个按钮上方显示比例数字（较大加粗），下方显示场景名（较小灰色）
- 选中状态：黑色填充背景 + 白色文字
- 未选中状态：白色背景 + 深色文字 + 浅灰色描边
- 圆角：约 10px
- 按钮等宽，间距约 8px

---

#### 2. 背景

**2.1 分类标签**

5 个胶囊形标签，水平排列：`全部` · `鲜艳` · `柔和` · `深色` · `自然`

- 选中（"全部"）：黑色填充 + 白色文字
- 未选中：无填充，浅灰色描边 + 深色文字

**2.2 随机选择按钮**

- 文字："✨ 随机选择壁纸"
- 样式：全宽，白色背景，浅灰描边，居中文字

**2.3 壁纸网格**

4 列 × 3 行共 12 张缩略图，圆角约 12px，间距 8px：

| 行  | 缩略图 1     | 缩略图 2     | 缩略图 3     | 缩略图 4     |
| --- | ------------ | ------------ | ------------ | ------------ |
| 1   | 粉玫瑰渐变 ✓ | 薰衣草白渐变 | 青绿渐变     | 粉橙紫渐变   |
| 2   | 绿黄橙渐变   | 橙色花卉抽象 | 蓝青渐变     | 紫薰衣草渐变 |
| 3   | 玫瑰花卉写真 | 深紫星云     | 深色彩虹油膜 | 深色极光雪景 |

- 已选中的缩略图（第 1 行第 1 个）右下角显示白色 **✓ 勾选徽章**

---

#### 3. 圆角半径

- 标签行：左对齐 "圆角半径"，右对齐当前值 **"16PX"**
- 滑块：黑色圆形滑块，当前位置约在轨道 1/3 处
- 端点标签：左 "直角"，右 "圆角"

---

#### 4. 摄像头

- **开关行**：Toggle（蓝色/开启状态） + 文字 "录制时显示摄像头画面"
- **大小控制**（Toggle 开启时显示）：
  - 标签行：左 "大小: 180px"
  - 滑块：黑色圆形滑块，当前位置约在轨道中间
  - 端点标签：左 "小"，右 "大"

---

#### 5. 麦克风

- 全宽下拉框
- 当前值："MacBook Pro 麦克风 (Built-in)"
- 右侧展示向下箭头图标 ▾
- 白色背景，浅灰描边，圆角约 8px

---

#### 6. 画布边距

- 标签行：左对齐 "画布边距"，右对齐当前值 **"48PX"**
- 滑块：黑色圆形滑块，当前位置约在轨道中间
- 端点标签：左 "无"，右 "大"

#### 8. 账户

- 未登录状态，显示一个全宽登录按钮
- 按钮文字："G 使用 Google 登录"

- 样式：白色背景，浅灰描边，居中内容

### 底部操作栏

两个按钮，水平排列，左右各占一半：

| 按钮     | 文字                        | 样式                               |

## 视觉规范总结

### 颜色

| 用途              | 描述                  |
| ----------------- | --------------------- |
| 页面背景          | 浅灰 `#F5F5F5`        |
| 面板背景          | 白色 `#FFFFFF`        |
| 主文字            | 深灰 `#1A1A1A`        |
| 次级文字          | 中灰 `#999999`        |
| 选中态            | 纯黑 `#000000`        |
| 描边              | 浅灰 `#E0E0E0`        |
| 付费按钮描边/文字 | 金橙色 `#F5A623` 附近 |
| 预览框边框        | 粉紫 → 绿渐变         |

### 圆角

| 元素           | 圆角大小        |
| -------------- | --------------- |
| 整体模态框     | ~16px           |
| 画面比例按钮   | ~10px           |
| 壁纸缩略图     | ~12px           |
| 下拉框         | ~8px            |
| 底部"完成"按钮 | ~12px（大圆角） |
| 颜色圆点       | 50%（正圆）     |

### 间距

- 模态框内边距：约 24px
- 各设置分组间距：约 20 ～ 24px
- 组内元素间距：约 12px

## 交互行为

| 操作                | 效果                                         |
| ------------------- | -------------------------------------------- |
| 切换画面比例        | 左侧预览框比例实时变化                       |
| 选择壁纸            | 预览背景实时更新，选中缩略图显示 ✓           |
| 拖动圆角滑块        | 预览框圆角实时变化                           |
| 拖动画布边距滑块    | 预览框内边距实时变化                         |
| 关闭摄像头 Toggle   | 隐藏"大小"滑块                               |
| 关闭光标高亮 Toggle | 隐藏颜色选择器（但颜色选择器在截图中仍显示） |
| 点击"随机选择壁纸"  | 从当前分类随机切换壁纸                       |
| 点击"完成"          | 保存设置，关闭模态框                         |
| 点击"×" 或遮罩      | 关闭模态框（不保存或保存取决于实现）         |
